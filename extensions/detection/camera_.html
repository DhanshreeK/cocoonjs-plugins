<static> cleanup(onSuccess, onFail, options)

Removes intermediate image files that are kept in temporary storage after calling camera.getPicture. Applies only when the value of Camera.sourceType equals Camera.PictureSourceType.CAMERA and the Camera.destinationType equals Camera.DestinationType.FILE_URI.
Parameters:
Name Type Description
onSuccess camera.onCleanupSuccess Called when the picture has been taken.
onFail camera.onFail Called when the picture has failed to be taken.
options string Camera options.
Example
navigator.camera.cleanup(onSuccess, onFail);

function onSuccess() {
    console.log('Camera cleanup success.');
}

function onFail(message) {
    alert('Failed because: ' + message);
}

CameraOptions

Properties:
Name Type Argument Default Description
quality number <optional>
 50 Quality of the saved image, expressed as a range of 0-100, where 100 is typically full resolution with no loss from file compression. (Note that information about the camera's resolution is unavailable.)
destinationType Camera.DestinationType <optional>
 Camera.DestinationType.FILE_URI Choose the format of the return value.
sourceType Camera.PictureSourceType <optional>
 Camera.PictureSourceType.CAMERA Set the source of the picture.
targetWidth number <optional>
 Width in pixels to scale image. Must be used with targetHeight. Aspect ratio remains constant.
targetHeight number <optional>
 Height in pixels to scale image. Must be used with targetWidth. Aspect ratio remains constant.
encodingType Camera.EncodingType <optional>
 Camera.EncodingType.JPEG Choose the returned image file's encoding.
mediaType Camera.MediaType <optional>
 Camera.MediaType.PICTURE Set the type of media to select from. Only works when PictureSourceType is PHOTOLIBRARY or SAVEDPHOTOALBUM.
allowEdit boolean <optional>
 false Allow simple editing of image before selection.
correctOrientation boolean <optional>
 false Rotate the image to correct for the orientation of the device during capture.
saveToPhotoAlbum boolean <optional>
 false Save the image to the photo album on the device after capture.
detection boolean <optional>
 false Enable Face features detection.
cameraDirection Camera.Direction <optional>
 Camera.Direction.BACK Choose the camera to use (front- or back-facing).
