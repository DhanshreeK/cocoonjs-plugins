NativeShare

Methods

<static> share(data, onSuccess, onFail)

Opens a given share native window to share some specific text content in any system specific social sharing options. For example, Twitter, Facebook, SMS, Mail, ...
Parameters:
Name Type Description
data Cocoon.NativeShare.ShareData The data to share
onSuccess Cocoon.NativeShare.onSuccess Called when the share has succeeded.
onFail Cocoon.NativeShare.onFail Called when the share has failed. In Android is not possible to know wether or not the user canceled the share dialog (http://stackoverflow.com/questions/3778048/how-can-we-use-startactivityforresult-for-email-intent) so only the success callback will be received.
Example
Cocoon.NativeShare.share({
                message: 'I have scored more points on Flappy Submarine!! Chooo choooo',
                image: 'url'
            }, function() {
                    console.log('Share completed');
            }, function(error) {
                    console.log('Share error: ' + error);
            });
