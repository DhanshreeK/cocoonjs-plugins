Welcome to the Canvas+ Extensions Docs!

Here you'll find detailed documentation for all of the Canvas+ extensions. Just select the specific extension on the left to open the relevant documentation section.

Also, remember that the extensions' source code is provided in this open repository, including access to future extensions code in our development branch.

We hope you find everything you need to get going here, but if you stumble on any problems with the docs or the extensions, just drop us a line at our Help Center and we'll do our best to help you out.
