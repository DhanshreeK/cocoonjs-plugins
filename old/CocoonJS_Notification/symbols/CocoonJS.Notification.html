{string} msg
The error message.
<static> CocoonJS.Notification.onPushNotificationDeliverySucceed(userData)
This CocoonJS.EventHandler object allows listening to events called when a notification is successfully delivered. The callback function receives a parameter with the notificationId of the delivered notification.

{string} userData
A string with the notification userData information.

This CocoonJS.EventHandler object allows listening to events called when a push notification is received. The callback function receives a parameter with the userData of the received notification.

{object} userData
An object with the notification userData information.
<static> CocoonJS.Notification.onRegisterForPushNotificationsFailed(msg)
This CocoonJS.EventHandler object allows listening to events called when the registration for push notifications fails. The callback function receives a parameter with error information.
Parameters:
{string} msg
The error message.
<static> CocoonJS.Notification.onRegisterForPushNotificationsSucceed(token)
This CocoonJS.EventHandler object allows listening to events called when the registration for push notification succeeds. The callback function does not receive any parameter.
Parameters:
{string} token
The token (apid or device token) received at registration time.

This CocoonJS.EventHandler object allows listening to events called when the unregistration for push notifications succeeds. The callback function does not receive any parameter.
