CocoonJS.EventHandler(nativeExtensionObjectName, cocoonJSExtensionObjectName, nativeEventName, chainFunction)
A class that represents objects to handle events. Event handlers have always the same structure: Mainly they provide the addEventListener and removeEventListener functions. Both functions receive a callback function that will be added or removed. All the added callback functions will be called when the event takes place. Additionally they also allow the addEventListenerOnce and notifyEventListeners functions.
Parameters:
{string} nativeExtensionObjectName
The name of the native extension object (inside the ext object) to be used.
{string} cocoonJSExtensionObjectName
The name of the sugarized extension object.
{string} nativeEventName
The name of the native event inside the ext object.
{number} chainFunction Optional
An optional function used to preprocess the listener callbacks. This function, if provided, will be called before any of the other listeners.
