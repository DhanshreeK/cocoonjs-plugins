<static> CocoonJS.App.proxifyConsole()
This function allows to forward console messages from the WebView to the CocoonJS debug console. What it does is to change the console object for a new one with all it's methods (log, error, info, debug and warn) forwarding their messages to the CocoonJS environment. The original console object is stored in the CocoonJS.App.originalConsole property.
