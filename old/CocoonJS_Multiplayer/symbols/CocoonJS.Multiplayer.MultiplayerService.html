Class Detail
CocoonJS.Multiplayer.MultiplayerService(nativeExtensionName, extensionName)
This type represents the access to a native Multiplayer extension API. This service allows to programmatically create matches to invite other players and to receive match invitations sent by other players. Your game must authenticate the related social service before you can use this service.
Parameters:
{string} nativeExtensionName
The name of the native ext object extension property name.
{string} extensionName
The name of the CocoonJS object extension property name.
Field Detail
<static> CocoonJS.Multiplayer.MultiplayerService.onInvitationReceived
This CocoonJS.EventHandler object allows listening to events called when an invitation has been received from another player and the user has accepted it The invitation will start loading. While the invitation is loading a system view might appear on the screen. You should pause/stop your current game and prepare it to process a new match
