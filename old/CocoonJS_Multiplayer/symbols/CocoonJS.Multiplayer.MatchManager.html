<static> CocoonJS.Multiplayer.MatchManager.onMatchFound(match, match)
This CocoonJS.EventHandler object allows listening to events called when a match has been found. The match might be returned before connections have been established between players. At this stage, all the players are in the process of connecting to each other. Always check the getExpectedPlayerCount value before starting a match. When its value reaches zero, all expected players are connected, and your game can begin the match. The callback function's documentation is represented by CocoonJS.Social.OnMatchFoundListener
Parameters:
{CocoonJS.Multiplayer.MatchManager} match
The source match manager
